# Plano de Implementação — Módulo FUNIS

## Visão Gera

Integrar o sistema de funis Kanban (referência: `ConexaoImplantes/funis-conexao`) como um módulo registrado dentro do ERP Conexão, seguindo os padrões arquiteturais existentes (ModuleDefinition, registries, React Query + Supabase, TanStack Router).

## Fase 1: Database — Migrations

**Arquivo:** `supabase/migrations/00029_funis.sql`

### Tabelas

```sql
-- 1. Funis (boards)
CREATE TABLE funis (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  titulo text NOT NULL,
  descricao text,
  created_by uuid REFERENCES auth.users(id) NOT NULL,
  empresa_id uuid REFERENCES empresas(id),
  created_at timestamptz DEFAULT now(),
  updated_at timestamptz DEFAULT now()
);

-- 2. Colunas do funil
CREATE TABLE funis_colunas (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  funil_id uuid REFERENCES funis(id) ON DELETE CASCADE NOT NULL,
  titulo text NOT NULL,
  posicao integer NOT NULL DEFAULT 0,
  created_at timestamptz DEFAULT now()
);

-- 3. Tarefas
CREATE TABLE funis_tarefas (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  funil_id uuid REFERENCES funis(id) ON DELETE CASCADE NOT NULL,
  coluna_id uuid REFERENCES funis_colunas(id) ON DELETE CASCADE NOT NULL,
  titulo text NOT NULL,
  descricao text,
  posicao integer NOT NULL DEFAULT 0,
  prioridade text CHECK (prioridade IN ('low','medium','high','urgent')) DEFAULT 'medium',
  atribuido_para uuid REFERENCES auth.users(id),
  tools text[] DEFAULT '{}',
  data_inicio date,
  data_fim date,
  depende_tarefa_id uuid REFERENCES funis_tarefas(id),
  parent_task_id uuid REFERENCES funis_tarefas(id),
  completed_at timestamptz,
  created_by uuid REFERENCES auth.users(id) NOT NULL,
  created_at timestamptz DEFAULT now(),
  updated_at timestamptz DEFAULT now()
);

-- 4. Permissões por funil (acesso granular)
CREATE TABLE funis_permissoes (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  funil_id uuid REFERENCES funis(id) ON DELETE CASCADE NOT NULL,
  user_id uuid REFERENCES auth.users(id) NOT NULL,
  nivel text CHECK (nivel IN ('view','edit')) NOT NULL DEFAULT 'view',
  created_at timestamptz DEFAULT now(),
  UNIQUE(funil_id, user_id)
);

-- 5. Índices
CREATE INDEX idx_funis_colunas_funil ON funis_colunas(funil_id, posicao);
CREATE INDEX idx_funis_tarefas_coluna ON funis_tarefas(coluna_id, posicao);
CREATE INDEX idx_funis_tarefas_funil ON funis_tarefas(funil_id);
CREATE INDEX idx_funis_permissoes_funil ON funis_permissoes(funil_id);
```

### RLS Policies (padrão ERP: usa `is_super_admin()` + `empresa_id`)

- `funis`: SELECT/INSERT/UPDATE/DELETE — super admin ou owner; empresa members veem seus funis
- `funis_colunas`: Herda do funil via `funil_id`
- `funis_tarefas`: Herda do funil via `coluna_id → funil_id`
- `funis_permissoes`: Apenas owner do funil pode gerenciar

### Trigger: auto-criar 4 colunas padrão no INSERT de funil

```sql
CREATE OR REPLACE FUNCTION criar_colunas_padrao_funil()
RETURNS TRIGGER AS $$
BEGIN
  INSERT INTO funis_colunas (funil_id, titulo, posicao) VALUES
    (NEW.id, 'Backlog', 0),
    (NEW.id, 'Em andamento', 1),
    (NEW.id, 'Revisão', 2),
    (NEW.id, 'Concluído', 3);
  RETURN NEW;
END;
$$ LANGUAGE plpgsql SECURITY DEFINER;

CREATE TRIGGER trg_funis_colunas_padrao
  AFTER INSERT ON funis
  FOR EACH ROW EXECUTE FUNCTION criar_colunas_padrao_funil();
```

### services/funis.ts — CRUD principal

Padrão: `supabase.from('funis').select/insert/update/delete` + React Query hooks no `useFunisData.ts`.

Funções:
- `listarFunis(empresaId?)` — lista funis do usuario (via RLS)
- `buscarFunil(id)` — busca funil completo (colunas + tarefas)
- `criarFunil(titulo, descricao)` — INSERT + trigger cria colunas
- `atualizarFunil(id, data)` — UPDATE titulo/descricao
- `deletarFunil(id)` — DELETE CASCADE

### services/tarefas.ts — CRUD + drag-drop

- `listarTarefasColuna(colunaId)` — tarefas ordenadas por posicao
- `criarTarefa(data)` — INSERT na coluna
- `atualizarTarefa(id, data)` — UPDATE campos
- `moverTarefa(tarefaId, novaColunaId, novaPosicao)` — UPDATE coluna_id + posicao
- `reordenarTarefas(colunaId, tarefaIds[])` — batch UPDATE posicoes
- `deletarTarefa(id)` — DELETE

### hooks/useFunisData.ts — React Query

```ts
// Queries
```

```ts
import { GitBranch } from 'lucide-react';
import { registerModule } from '~/registry/modules';
import { registerNavItem } from '~/registry/nav-items';

export const funisModule = {

  icon: GitBranch,

    { key: 'funil.atualizado', label: 'Funil Atualizado', descricao: 'Quando um funil e editado', type: 'status_change' },
    { key: 'funil.excluido', label: 'Funil Excluido', descricao: 'Quando um funil e removido', type: 'status_change' },

      moduloKey: 'funis-conexao',
```
