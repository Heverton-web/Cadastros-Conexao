# Renomear Nav Items do Hub para nomes únicos

Labels duplicados no sidebar: "Dashboard" (4x), "Conquistas" (3x), "Ranking" (2x), "Analytics" (2x).

| Atual | Novo |
|-------|------|
| Dashboard (admin) | Dash Admin |
| Materiais | Add Materiais |
| Trilhas | Add Trilhas |
| Badges | Add Badges |
| Analytics (admin) | BI Admin |
| Chatbot | Config. Chatbot |
| Dashboard (gestor) | Dash Gestor |
| Analytics (gestor) | BI Gestor |
| Ranking (gestor) | Rank Gestor |
| Conquistas (gestor) | Badges Gestor |
| Dashboard (consultor) | Dash Consultor |
| Ranking (consultor) | Rank Consultor |
| Conquistas (consultor) | Badges Consultor |
| Dashboard (distribuidor) | Dash. Distribuidor |
| Conquistas (distribuidor) | Badges Distribuidor |

- `src/features/hub/module.ts` — linhas 62-210 (15 `registerNavItem`)
