## Praticas Essenciais Linux
1. Gerencie pacotes com apt/yum/dnf — evite instalacoes manuais.
2. Use systemctl para gerenciar servicos systemd.
3. Permissoes: principio do menor privilegio (chmod 755/644, nunca 777).
4. Logs em /var/log — use journalctl para logs systemd.
5. Monitore com top/htop, df -h, free -m, netstat/ss.
6. Automatize tarefas com cron e systemd timers.
7. Use ssh-keygen + ssh-copy-id para acesso SSH sem senha.
8. Configure firewall com ufw ou iptables.
9. Backup: rsync, tar, e scripts agendados.
10. Documente tudo em scripts e playbooks Ansible.
