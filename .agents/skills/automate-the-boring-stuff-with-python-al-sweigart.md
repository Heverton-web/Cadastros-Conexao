Instalando o pip O arquivo executável da ferramenta pip chama-se pip no Windows e pip3 no macOS e Linux. A ferramenta pip está incluída no Python desde a versão 3.4. No entanto, algumas distribuições Linux podem não a ter pré-instalada. Para instalar o pip3 no Ubuntu ou Debian Linux, abra uma nova janela do terminal e digite `sudo apt install python3-pip`. Encontrando pip Se a pasta do pip não estiver incluída na variável de ambiente PATH, talvez seja necessário mudar de diretório no terminal com o comando `cd` antes de executar o pip. Primeiro, descubra seu nome de usuário executando `echo %USERNAME%` no Windows ou `whoami` no macOS e Linux. Em seguida, no Windows, execute o seguinte comando, especificando seu nome de usuário e ajustando o nome da pasta de acordo com a versão do Python que você instalou: C:\Users\al&gt; cd C:\Users\ your_username \AppData\Local\Programs\Python\Python313\Scripts No macOS, execute este comando (certificando-se de especificar o nome da pasta correto): al@Als-MacBook-Pro ~ % cd /Library/Frameworks/Python.framework/Versions/3.13/bin/ No Linux, execute este comando, especificando seu nome de usuário: al@al-VirtualBox:~$ cd /home/ your_username /.local/bin/ Você agora deve estar na pasta correta para executar a ferramenta pip. Como alternativa, você pode adicionar essas pastas à variável de ambiente PATH seguindo as instruções em “A variável de ambiente PATH”, na página 261 do Capítulo 12. Executando o pip a partir de ambientes virtuais Alguns sistemas operacionais podem não permitir a execução do pip sem antes criar e ativar um ambiente virtual usando o módulo venv do Python.
