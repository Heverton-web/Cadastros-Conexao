# RTK Memory

Sempre que corrigir um bug de compilação (ex: tipagem do Vitest ou conexões do Docker), execute a ferramenta de edição de arquivo e adicione a regra exata descoberta na seção `RTK SCRATCHPAD` do arquivo `agents.md` na raiz do projeto. Isso evitará que você precise reanalisar o erro no futuro.
