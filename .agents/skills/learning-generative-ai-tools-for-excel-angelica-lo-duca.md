Macros e VBA Uma macro no Excel é um conjunto de comandos automatizados que permite executar tarefas repetitivas rapidamente, como formatar dados, criar relatórios ou realizar cálculos complexos. Você pode escrever macros usando a linguagem de programação VBA (Visual Basic for Applications) ou gravar macros sem escrever código. Antes de usar macros, você precisa ativá-las seguindo estes passos (no Windows), caso ainda não estejam ativadas: Abra o Excel e vá em Arquivo → Opções. Selecione a opção Desativar macros VBA com notificação (recomendado). Clique em OK para confirmar as alterações. Se você usa Mac OS, siga estes passos: Abra o Excel e vá para Excel → Preferências → Segurança → Desative todas as macros com notificação. Existe uma diferença fundamental entre o conceito de função e o de fórmula: As funções são operações predefinidas que simplificam cálculos complexos ou repetitivos, como somar números, calcular médias ou procurar valores específicos (por exemplo, SUM, AVERAGE ). As fórmulas são expressões definidas pelo usuário que realizam cálculos ou manipulações em dados. Elas sempre começam com um sinal de igual ( = ) e podem combinar operadores, constantes e referências (como =A1+A2 ). No entanto, pelo menos dois casos indicam que seu uso ainda não está maduro (pelo menos até a data de publicação deste livro): Grandes conjuntos de dados Ao trabalhar com grandes conjuntos de dados, a IA pode encontrar algumas dificuldades em relação ao desempenho e à precisão.
