O que é o ERP Conexão?

O ERP Conexão é uma plataforma SaaS multiempresa desenvolvida para o ecossistema Conexão Implantes. A aplicação centraliza a gestão de cadastros de profissionais da saúde (cirurgiões-dentistas, distribuidores e consultores), relacionamento com clientes (CRM), pesquisas de satisfação (NPS), mapas interativos de cobertura, kanban de processos (Funis), links inteligentes (LinkTree) e uma plataforma completa de treinamento e educação (Hub).

O sistema foi projetado com arquitetura modular e permissionamento granular, permitindo que cada empresa cliente opere de forma isolada dentro da plataforma, com suas próprias regras de negócio, módulos ativos e usuários.

Objetivos de Negócio

  • Centralização — Unificar a gestão de profissionais, distribuidores e consultores em uma única plataforma
  • Isolamento Multiempresa — Cada organização opera com dados e configurações próprias, sem risco de vazamento entre tenants
  • Automação de Processos — Fluxos de aprovação de cadastros, documentos e campos com notificações em tempo real
  • Inteligência de Negócio — BI e dashboards com métricas de desempenho comercial e satisfação de clientes
  • Educação Continuada — Plataforma de treinamento (Hub) com trilhas, materiais e gamificação para distribuidores e consultores
  • Relacionamento com Cliente — CRM completo com gestão de carteira, visitas, transferências e diretoria

Stack Tecnológica

Frontend — TanStack Start (React Router) + Vite + Tailwind CSS v4 • Backend — Supabase (PostgreSQL, Auth, Storage, RLS) • Linguagem — TypeScript (TSX) • Ícones — Lucide React • Deploy — Docker + Docker Swarm na VPS
  • TanStack Start / React Router — Framework de aplicação com roteamento baseado em arquivos, renderização SSR e server functions
  • Vite — Build tool com HMR rápido e otimização de produção
  • Supabase — Banco PostgreSQL, autenticação (Auth), armazenamento (Storage) e políticas de segurança em linha (RLS)
  • Tailwind CSS v4 — Estilização utilitária com design system customizado
  • Zustand — Gerenciamento de estado global
  • Docker — Containerização e orquestração com Docker Swarm

Público-Alvo

  • Empresas (distribuidoras de implantes) — utilizam o sistema para gerir seus cadastros e operações
  • Consultores — profissionais de campo que utilizam o CRM e o Hub
  • Distribuidores — parceiros com acesso a mapas e métricas regionais
  • Gestores e Diretores — acompanham indicadores via dashboards e BI
  • Super Administradores — equipe técnica da Conexão que gerencia toda a plataforma
