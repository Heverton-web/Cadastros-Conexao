Capítulo 6
Módulos Exclusivos do Super Admin

Painéis e ferramentas de administração global da plataforma ERP Conexão.

⚠ Acesso restrito a usuários com perfil is_super_admin = true

O Super Admin é o perfil de mais alta autoridade no sistema. Usuários com esta flag recebem automaticamente todas as permissões do sistema e têm acesso a uma seção exclusiva de administração global, com 12 itens de navegação no menu "Administração".

Atenção: A seção de Administração só aparece no menu de usuários Super Admin. Usuários comuns nunca veem estas rotas ou funcionalidades.

Painéis de Administração Global

⚙ Módulos ERP

Gerencia todos os módulos da plataforma: ativar/desativar módulos globalmente, configurar parâmetros de cada módulo e visualizar o status de cada um.

Super Admin Only
  • /global/modulos — Lista de módulos
  • /global/modulos/$key — Detalhe do módulo

☷ Empresas ERP

Gerencia as empresas cadastradas na plataforma: criar, editar, ativar/desativar empresas, configurar módulos ativos por empresa e visualizar detalhes de cada tenant.

Super Admin Only
  • /global/empresas — Lista de empresas
  • /global/empresas/$id — Detalhe da empresa

☼ Dashboard NPS Global

Visão consolidada de todas as pesquisas NPS de todas as empresas. Permite comparar métricas de satisfação entre tenants.

Super Admin Only
  • /global/nps — Dashboard NPS global

☯ Dashboard Hub Global

Visão consolidada da plataforma de treinamento Hub em todas as empresas. Métricas de uso, materiais, trilhas e engajamento.

Super Admin Only
  • /global/hub — Dashboard Hub global

⚛ Permissões

Gerencia o sistema de permissões global: visualiza todas as permissões registradas, seus grupos e descrições. Permite auditoria do sistema de permissões.

Super Admin Only
  • /global/permissoes — Central de permissões

⚠ Central de Ações

Monitora e gerencia ações automáticas e webhooks do sistema. Visualiza histórico de execuções, filas e logs de ações.

Super Admin Only
  • /global/acoes — Central de ações

☰ Banco de Dados

Ferramenta de visualização e gestão do banco de dados Supabase. Permite consultar tabelas, visualizar schemas e monitorar o banco.

Super Admin Only
  • /global/banco — Banco de dados

☍ Integrações Nativas

Gerencia integrações com sistemas externos: PayTrack, webhooks, API connectors e outras integrações nativas da plataforma.

Super Admin Only
  • /global/integracoes — Integrações nativas

⚡ Credenciais Demo

Gerencia credenciais de demonstração do sistema. Permite criar, editar e remover contas demo para apresentações e testes.

Super Admin Only
  • /global/demos — Credenciais demo

☮ Laboratório de Testes

Ambiente isolado para testes de funcionalidades, experimentos e validações antes de liberar para produção.

Super Admin Only
  • /global/laboratorio — Laboratório

☢ Limites de Credenciais

Gerencia limites de credenciais de acesso ao sistema por empresa ou usuário. Controla quotas e restrições de acesso.

Super Admin Only
  • /global/limits — Limites de credenciais

🎨 Temas Hub

Configura temas visuais da plataforma Hub para todas as empresas. Gerencia paletas, layouts e personalização da plataforma de treinamento.

Super Admin Only
  • /empresa/hub/tema — Temas Hub

Resumo do Acesso Super Admin

Total de rotas exclusivas: 14 rotas administrativas em 12 categorias
Permissões: Todas as 86+ permissões do sistema concedidas automaticamente
Visibilidade: Seção "Administração" no menu lateral, visível apenas para Super Admin
Gerência: Empresas, módulos, permissões, ações, banco, integrações, demos, laboratório e limites
